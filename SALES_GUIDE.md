## 🔑 ライセンスキーの発行方法

### 方法①：管理WebUI（推奨）

1. **https://shadowing-web.onrender.com/admin** にアクセス
2. 管理者パスワードでログイン
3. 「🔑 キー発行」タブを開く
4. 単品またはバンドルセットを選択して発行
5. 表示されたキーをコピーして購入者に送信

### 方法②：Render.comのShell（CLIで発行）

Render.com → shadowing-web → Shell から：

```bash
# 単品発行
node tools/gen-license.mjs base 1 "Booth注文#1234"
node tools/gen-license.mjs pack:travel_vol1 1 "Booth注文#1234"

# バンドル発行（一括）
node tools/gen-license.mjs bundle:travel_set 1 "Booth注文#1234"
node tools/gen-license.mjs bundle:toeic_set 1 "Booth注文#1234"

# ライセンス一覧確認
node tools/admin.mjs list
```

## 🛍️ Boothでの販売設定

### 推奨プラットフォーム：Booth（pixiv運営）
- URL: https://booth.pm
- 手数料：売上の10%（クレカ決済は別途3.6%）
- デジタルデータ（ダウンロード形式）で販売可能

### 商品設定の手順

1. **Boothのshopを開設**（https://booth.pm）
2. 各商品ページを作成：
   - 商品名・説明文を入力
   - ダウンロードファイルとして「使い方・ライセンス発行手順.txt」を添付
   - ※実際のキーは受注後に手動で送付（またはBoothのメッセージ機能で）

3. **注文が入ったら**：
   - 管理画面（/admin）でキーを発行
   - 「📧 配信テキストをコピー」でテンプレートを生成
   - Boothのメッセージ or メールで購入者に送付

> **補足**: Boothには「ダウンロードコンテンツ」として事前にキーを含むテキストファイルを登録する方法もありますが、在庫管理が手間です。受注後に発行する方式が確実です。

## ➕ 新しいパックを追加する方法

別売りパックは将来いくつでも追加できます。

### 手順

1. `/packs/[パックID].json` を作成（下記フォーマット参照）
2. GitHubにpush → Render.comが自動デプロイ
3. 管理画面でパックキーを発行して販売開始

### パックのJSONフォーマット

```json
{
  "id": "パックID（英数字、例: business_vol1）",
  "name": "表示名（例: ビジネス英語パック vol.1）",
  "icon": "絵文字（例: 💼）",
  "description": "説明文",
  "categories": [
    { "id": "cat1", "label": "カテゴリ名", "icon": "💬" }
  ],
  "texts": [
    {
      "id": 数値（かぶらないように）,
      "category": "cat1",
      "title": "English Title",
      "titleJa": "日本語タイトル",
      "text": "A: ...\nB: ...\nA: ..."
    }
  ],
  "explanations": {
    "テキストID": [
      { "phrase": "重要フレーズ", "reading": "", "meaning": "意味", "note": "補足" }
    ]
  }
}
```

### パックIDとキー

| パックID（ファイル名） | ライセンス商品コード |
|----------------------|-------------------|
| `business_vol1.json` | `pack:business_vol1` |
| `medical.json` | `pack:medical` |
| `kids.json` | `pack:kids` |

管理画面の「キー発行」でカスタム商品コードを入力すれば発行できます。

## 🔒 セキュリティ設定

### Render.comで設定する環境変数

| 変数名 | 内容 |
|--------|------|
| `ADMIN_PASSWORD` | 管理画面のパスワード（必須） |
| `DATA_DIR` | `/data`（設定済み） |

### ADMIN_PASSWORDの設定方法
1. https://dashboard.render.com → `shadowing-web` サービス
2. **Environment** タブ → **Add Environment Variable**
3. Key: `ADMIN_PASSWORD` / Value: 強力なパスワードを設定
4. **Save Changes** → 自動再デプロイ
