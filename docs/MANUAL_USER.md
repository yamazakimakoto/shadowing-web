# ユーザーマニュアル — 英会話シャドウイング（モデル版）

## はじめに

英会話シャドウイング練習アプリへようこそ。ネイティブの音声を聞きながら後追い音読（シャドウイング）することで、英語の発音・リズム・スピードを身につけるアプリです。

---

## はじめてご利用の方（初期設定）

### 1. アプリを開く

ブラウザで以下のURLにアクセスしてください：

```
https://shadowing-web.onrender.com
```

### 2. ライセンスキーを入力する

初回アクセス時にライセンスキーの入力画面が表示されます。

1. 購入時にお送りしたライセンスキー（`SHAD-XXXX-XXXX-XXXX` 形式）を入力
2.「認証する」ボタンをタップ
3. 認証が完了するとアプリが起動します

> **注意:** 1つのキーは最大5台のデバイスで使用できます。

---

## 基本的な使い方

### テキストを選ぶ

1. 上部のカテゴリタブ（「日常」「旅行」「ビジネス」など）から練習したい分野を選ぶ
2. 一覧からテキストをタップして選択
3. 選択したテキストが画面中央に表示されます

### 音声を再生する

| ボタン | 動作 |
|--------|------|
| **▶ 再生** | 音声を再生する |
| **⏸ 一時停止** | 再生を一時停止する |
| **■ 停止** | 再生を停止する |
| **🔁 リピート** | 繰り返し再生をON/OFFする |

### 再生速度を調整する

「速度」スライダーで再生速度を調整できます。
- `0.5` → ゆっくり（初心者向け）
- `1.0` → 標準速度
- `1.5` → 速め（上級者向け）

### 音声の種類を変える

「音声」選択ボックスから好みの音声を選択できます：

| 音声名 | 特徴 |
|--------|------|
| **Nova** | 女性・明瞭（おすすめ） |
| **Shimmer** | 女性・柔らかめ |
| **Alloy** | 中性的 |
| **Echo** | 男性・自然 |
| **Fable** | 英国系アクセント |
| **Onyx** | 男性・低めの声 |

## 会話練習モード（ダイアログ）

`A:` `B:` のある会話形式テキストでは、役割を分けた練習ができます。

### 全体読み上げ

「全体」ボタンを選択すると A・B 両方のセリフを順番に読み上げます。音声を聞きながら全体の流れを把握するのに使います。

### 役割練習（シャドウイング）

1. 「練習」ボタンを選択
2. 練習したいパート（A または B）を選択
3. 再生すると相手のセリフだけ読み上げられ、自分のパートは無音になります
4. 無音の間に自分でそのセリフを声に出して練習

## 追加パックの購入・登録

デフォルトの100題以外に、専門分野のパックを追加できます。

### 対応パック

| パック名 | 内容 |
|---------|------|
| ホテル英会話パック | チェックイン・レストラン・トラブル対応など |
| TOEICパック | TOEIC頻出シーン・ビジネス英語 |
| 旅行編 vol.1 | 空港・ホテル・交通 |
| 旅行編 vol.2 | 観光・ショッピング・レストラン |
| 自字 vol.1/vol.2 | 日常・文化・ニュース |

### パックの登録手順

1. 下部ナビの「パック」タブを開く
2.「パックキーを入力」をタップ
3. 購入時にお送りしたパック専用キーを入力
4.「登録する」ボタンをタップ
5. パックが追加されてテキスト一覧に表示されます

## 便利な機能

### テキストを手動入力

「テキスト入力」タブに任意の英文を貼り付けて練習できます。教科書や教材の文章をそのまま使えます。

## よくある質問

**Q: 音声が出ない**
A: ブラウザの音声許可設定を確認してください。Safari の場合、設定 → Safari → マイクとマイクを「許可」にしてください。

**Q: ライセンスキーを入力しても認証されない**
A: キーの大文字・ハイフンを正確に入力してください。コピー＆ペーストが確実です。同じキーが既に5台で使用されている場合は管理者にご連絡ください。

**Q: 録音・採点機能はありますか？**
A: 現在準備中です。料金プランの整備後に提供開始予定です。

**Q: オフラインで使えますか？**
A: OpenAI音声再生はインターネット接続が必要です。一度ライセンス認証を行えば、Web Speech API による音声再生（OS標準音声）は通信なしで利用できます。

**Q: 端末を変えた場合は？**
A: 新しい端末で同じキーを入力してください（最大5台まで）。
