➕ 新パック追加の手順

1. /packs/[パックID].json を作成する

2. git add -A && git commit -m "〇〇パック追加" && git push

3. Render.com が自動デプロイ → サーバー再起動時にパックが /data/packs/ にコピーされる

4. 管理画面でパックキーを発行して販売開始

⚠️ データ消失の原因と恒久対策

ライセンスが消える主な原因：Render.com の永続ディスクが未接続

1. dashboard.render.com → shadowing-web → Disks タブを確認

2. shadowing-data ディスクが マウント済み（/data） になっているか確認

3. ない場合 → Add Disk（Name: shadowing-data / Mount Path: /data / Size: 1GB）
